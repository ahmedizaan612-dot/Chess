# Python Chess Engine ♟️

A fully functional chess engine built from scratch in Python using Pygame.

This project started as a learning challenge and evolved into a complete chess system with real chess rules, move validation, simulations, and special move handling.

---

## Features

✅ Full legal chess movement
✅ Piece capture system
✅ Check detection
✅ King safety validation
✅ Move simulation system
✅ Castling (Kingside & Queenside)
✅ En Passant
✅ Click-based movement system
✅ Attack square checking
✅ Illegal move prevention
✅ Custom board and piece rendering
✅ Turn-based gameplay system

---

## Built With

* Python
* Pygame

---

## Project Highlights

### Simulation-Based Move Validation

The engine uses simulation logic to prevent illegal moves and properly validate king safety.

### Full Chess Rules

Special chess rules like:

* Castling
* En Passant
* Check detection

were implemented manually without using external chess libraries.

### Self-Built Logic System

All movement systems, move legality checks, attack detection, board state management, and special move logic were designed from scratch.
